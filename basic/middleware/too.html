Token Verification Middleware

This demo simulates Express.js token verification middleware. Try entering different tokens to see how the middleware responds.

// Token verification middleware
app.use("/api", (req, res, next) => {
  // Check if token is provided and valid
  if (req.query.token === '123') {
    next(); // Token is valid, proceed to route handler
  } else {
    res.status(401).send('Unauthorized'); // Invalid token
  }
});

Try these values:

Valid token: 123 | Invalid tokens: abc, 456, or leave empty

This simulates a request to: GET /api/data?token=YOUR_INPUT
